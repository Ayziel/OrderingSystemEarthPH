User Manual for Agent Order Form

Introduction

The Agent Order Form is a web-based tool used for recording order details submitted by agents. It ensures accurate and efficient collection of key information about the store, agent, and order specifics.


Accessing the Form

  • Open your web browser.
  • Go to Agent Order Form.
  • The form will load, displaying multiple input fields for agent, store, and order information.


Field Descriptions and Instructions

Agent Name

  • What it is: The name of the agent filling out the form.
  • How to fill: Type the agent’s full name in the provided field.

Team Leader Name

  • What it is: The name of the agent's team leader.
  • How to fill: The name of the agent's team leader.

Area

  • What it is: The geographic location where the store is located.
  • How to fill: Specify the area name or location.

Order Date

  • What it is: The date when the order is being placed.
  • How to fill: Select the current date or type it in the format (MM/DD/YYYY).

Store Name

  • What it is: The name of the store placing the order.
  • How to fill: Enter the store's official name.

House Number and Street

  • What it is: The store’s complete address details, including house/building number and street name.
  • How to fill: Input the house number and street name accurately.

Barangay

  • What it is: The barangay where the store is located.
  • How to fill: Enter the barangay name.

Store Code

  • What it is: A unique code assigned to the store.
  • How to fill: Input the store’s official code. If unsure, contact the team leader.

TIN (Tax Identification Number)

  • What it is: The tax identification number assigned to the store.
  • How to fill: Input the TIN number accurately.


Navigating the Form

Once all required fields are filled, click the "Next" button to proceed. If any required fields are missing, an error message will prompt you to complete them.


Common Issues and Troubleshooting

  • Form Not Loading: Check your internet connection and reload the page.
  • Input Errors Ensure all required fields are completed properly.
  • Incorrect Format for Dates: Use the MM/DD/YYYY format for date entries.


Tips for Accurate Submission

  • Double-checkc all input details before clicking "Next".
  • Ensure the TIN and store code are correct as these are essential for tracking purposes.


Frequently Asked Questions (FAQs)

    Q1: What if I don't know the store’s TIN?A1: Contact the team leader or store owner for the correct TIN.

    Q2: Can I edit the form after submission?A2: No, ensure all details are accurate before clicking "Next".

    Q3: What happens if I enter the wrong date?A3: Return to the Order Date field and enter the correct date before proceeding.



Contact for Support

If you encounter issues that cannot be resolved, please contact the support team at support@example.com or reach out to your team leader for assistance.
